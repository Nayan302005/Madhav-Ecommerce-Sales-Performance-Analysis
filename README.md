📌 Overview

This project provides a data-driven visualization of ecommerce performance using Power BI. It helps stakeholders understand:

💰 Revenue & Profit Trends
📦 Sales Quantity Distribution
🧾 Customer Purchasing Patterns
🌍 State-wise Performance
💳 Payment Mode Insights
🎯 Key Features

✨ Dynamic Dashboard Filters

Quarter-wise filtering (Q1, Q2, Q3, Q4)
State-based filtering for regional insights

🧠 Insights Derived
🏆 Top Performing State: Maharashtra
💳 Most Used Payment Mode: COD (44%)
📦 Highest Selling Category: Clothing (63%)
📈 Best Profit Month: November
📉 Loss Months Identified: May & July
